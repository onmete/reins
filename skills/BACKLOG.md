# Skills Backlog

Potential improvements to track. Not loaded by agents.

## All upstream skills

- **Confluence publication** — when Confluence MCP write tools
  are available, publish feature specs, decomposition records,
  and spike findings as Confluence pages linked from Jira.
  Replaces or supplements comment-based linking.
- **Template customization** — allow teams to override
  artifact formats (feature spec, Epic/Story templates) via
  `.reins/templates/`.

## reins-decompose

- **Parallel story creation** — if Jira MCP supports batching,
  create all stories in parallel instead of sequentially.
- **Auto-assign** — if team member info is available, suggest
  assignments based on component ownership.

## reins-spike

- **Research MCP** — when a dedicated research MCP (e.g.
  Perplexity) is available, use it for deeper synthesis and
  citation-backed answers instead of WebSearch + WebFetch.
- **Prototype scaffolding** — option to generate a throwaway
  prototype branch for spikes that need hands-on validation,
  with automatic cleanup after the spike concludes.
